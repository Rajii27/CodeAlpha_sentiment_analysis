# Amazon Fine Food Reviews Sentiment Analysis

## Project Overview

This project performs Sentiment Analysis on the Amazon Fine Food Reviews dataset using Natural Language Processing (NLP). Customer reviews are analyzed and classified into Positive, Negative, and Neutral sentiments using the VADER Sentiment Analyzer. Various visualizations are created to understand customer opinions, review patterns, and overall customer satisfaction.

---

## Tasks Performed

- Data Loading
- Data Exploration
- Data Cleaning
- Missing Value Handling
- Duplicate Removal
- Text Preprocessing
- Sentiment Analysis using VADER
- Data Visualization
- Customer Review Analysis

---

## Tools Used

- Python

- NLTK (VADER Sentiment Analyzer)
- WordCloud

---

## Dataset Description

The dataset contains Amazon Fine Food Reviews with the following information:

- Product ID
- User ID
- Profile Name
- Helpfulness Score
- Review Score
- Review Summary
- Review Text

---

## Visualizations Created

- Sentiment Distribution
- Sentiment Percentage (Pie Chart)
- Customer Rating Distribution
- Rating vs Sentiment
- Review Length Distribution
- Positive Review Word Cloud
- Negative Review Word Cloud

---

## Key Findings

- Most customer reviews express positive sentiment, indicating overall customer satisfaction.
- Higher-rated products generally receive positive reviews.
- Lower ratings are commonly associated with negative sentiments.
- Most customers write short to medium-length reviews.
- Positive reviews frequently mention product quality, taste, and satisfaction.
- Sentiment analysis helps businesses understand customer feedback and improve products.

---

## Conclusion

This project demonstrates the application of Natural Language Processing (NLP) for analyzing customer reviews and identifying sentiment. By combining data preprocessing, VADER sentiment analysis, and data visualization, meaningful insights were extracted from Amazon Fine Food Reviews. The project highlights how sentiment analysis can support business decisions by understanding customer opinions and product performance.
